Object-oriented programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm in computer science that relies on the concept of classes and objects.

The implementation of OOP varies throughout programming languages - but the pillars (Abstraction, Inheritance, Encapsulation, and Polymorphism) of OOP are usually the same.

Freecodecamp offers a great crash course about OOP without using any specific programming language

Depending on which track or programming language you decided to learn, you should pick one of those materials to learn OOP from:

  • C++ (chapter 14)
  • Java (head first object-oriented analysis and design is one of the best materials you can find about the concept of object oriented design)
  • JavaScript
