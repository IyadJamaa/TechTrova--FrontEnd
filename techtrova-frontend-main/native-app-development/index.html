IOS Development

In order to start your career as a IOS developer you first need to have access to a Mac device as most of the tools you need to learn are only available on MacOS, native IOS development is also a really vast field with many new opportunities especially with the new introduction of the Apple Vision Pro, which open a new fields for developers to explore.




Initial Step

You can develop IOS applications using two programming languages and two UI frameworks they are :

• Swift and SwiftUI (which are relatively new)

• Object C and UIkit (which most legacy IOS apps are using)



If you are developing a new application most of the time you will use Swift and SwiftUI but in some cases you might need to use UIkit with them, so it’s better to start learning swift and swiftUI and gradually learn object C and UIkit.
